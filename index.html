Jawab :

Cara kerja website : Website berfungsi ketika pengguna memasukkan alamat domain, browser mengonversi ke alamat IP melalui server DNS, dan terhubung ke web server. Server memproses permintaan, mengirimkan halaman web, yang ditampilkan di browser, memungkinkan akses dan interaksi dengan konten melalui internet.

Hubungan : Web server menyimpan dan mengelola konten situs web. Domain digunakan sebagai alamat unik untuk mengakses situs web tersebut. Hosting menyediakan ruang server dan sumber daya untuk menyimpan dan membuat situs web dapat diakses secara online. Dengan kerjasama ketiganya, pengguna dapat mengakses suatu situs web dengan memasukkan domain di browser mereka. Domain diarahkan ke alamat IP yang sesuai dengan web server, dan konten situs disajikan melalui hosting.
